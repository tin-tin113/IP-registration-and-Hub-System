1. Executive Summary

The CHMSU IP Registration and Hub System is a web-based application that digitizes intellectual property management at Carlos Hilado Memorial State University. It provides online submission, multi-stage verification, automated certificate generation, and a public IP Hub.

2. System Overview

2.1 System Modules

Module Description Primary Users
IP Registration Portal Online submission of Copyright, Patent, Trademark Users
Verification Dashboard Document and payment verification Clerks, Directors
Approval System Final approval with certificate generation Directors
IP Hub Repository Public showcase of approved IP works Public
Analytics Dashboard System-wide statistics Directors
